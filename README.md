# Data Analysis of IMDB Movie Dataset (2023-1951)

## Overview
Comprehensive EDA, visualizations, statistical tests, and linear regression on IMDB Movie Dataset.

**Features:**
- Data cleaning & preprocessing
- Histograms, scatter plots, box plots
- Top genres, directors, actors, genre combinations
- Movies per decade
- Linear regression (decade vs movie count)
- T-test & Chi-square test

## Requirements
```bash
pip install pandas numpy matplotlib seaborn scikit-learn scipy
```

## Usage
```bash
python Data_Analysis_of_IMDB-Movie-Dataset(2023-1951).py
```

## Dataset
[IMDB-Movie-Dataset(2023-1951).csv](IMDB-Movie-Dataset(2023-1951).csv) - Columns: Movie ID, Name, Year, Genre, Overview, Director, Cast

## Visualizations Generated
- Year distribution histogram
- Year vs movies scatter with trendline
- Year box plot
- Top 10 genres (bar & pie)
- Movies per decade bar
- Top directors bar
- Top genre pairs bar & heatmap
- Top actors bar
- Linear regression plot
